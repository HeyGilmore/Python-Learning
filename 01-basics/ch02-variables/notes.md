## Key Terms

### Variable
A named location that stores a value which can be used or changed throughout a program.

### String
A series of characters enclosed in single or double quotes.

### Method
An action Python can perform on a piece of data.

### Traceback
A report generated by Python when it encounters an error. It helps identify where the problem occurred.

### Integer
A whole number without a decimal point.

### Float
A number containing a decimal point.

## In My Own Words

- When a Python file runs, the Python interpreter reads the code from top to bottom and executes each instruction.
- A variable acts like a labeled container that holds information.
- Variables can contain letters, numbers, and underscores, but cannot start with a number.
- Variable names should be descriptive and easy to understand.
- Most programming errors come from small mistakes such as typos or missing characters.
- A traceback helps locate the source of an error.
- Methods allow Python to manipulate data in different ways.
- Strings can be modified using methods like `upper()`, `lower()`, and `title()`.
- Special characters such as `\n` and `\t` help format output.
- Integers are whole numbers, while floats contain decimal values.
- The `str()` function converts numbers into strings so they can be displayed with text.
- Comments make code easier to understand and maintain.
