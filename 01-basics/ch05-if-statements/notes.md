## Most Important Takeaways

1. Programs make decisions using conditional statements.
2. Every conditional test returns either `True` or `False`.
3. `==` checks equality, while `=` assigns a value.
4. `if` executes code only when a condition is true.
5. `if-else` provides two possible execution paths.
6. `if-elif-else` allows multiple conditions to be checked.
7. `and`, `or`, and `not` combine or modify conditions.
8. Membership operators (`in` and `not in`) check whether values exist in lists.
9. Empty lists evaluate to `False`.
10. Clear indentation is essential for readable and correct Python code.
